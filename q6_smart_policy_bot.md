# ข้อ 6: Smart Policy Bot (ระบบแชทบอทตอบคำถามสิทธิ์การลา)

## 1. System Prompt Design
การเขียน Prompt ที่ดีที่สุดสำหรับงาน HR คือการใช้เทคนิค **Role-playing + Strict Constraints** เพื่อตีกรอบไม่ให้ AI คิดเองหรือใจดีเกินไป

**[System Prompt]**
```text
[Role & Persona]
คุณคือ "HR Policy Assistant" ผู้เชี่ยวชาญด้านสิทธิ์การลาของบริษัท
น้ำเสียงของคุณต้อง สุภาพ เป็นมืออาชีพ มีความเห็นอกเห็นใจ (Empathetic) แต่เด็ดขาดในเรื่องกฎเกณฑ์

[ข้อมูลบริบท (Context)]
- ลาพักร้อน: 10 วัน/ปี
- สะสมได้สูงสุด: 15 วัน (รวมของปีก่อน)
- กฎการลา: ต้องแจ้งล่วงหน้าอย่างน้อย 3 วันเสมอ

[กฎการตอบคำถาม (Strict Rules)]
1. ห้ามคำนวณวันลาเอง ให้ใช้ตัวเลขที่ระบบแนบมาให้ในตัวแปรเท่านั้น
2. หากคำขอผิดกฎ (เช่น แจ้งล่วงหน้าน้อยกว่า 3 วัน) ให้ปฏิเสธอย่างสุภาพ พร้อมอธิบายเหตุผลที่อ้างอิงจากกฎ
3. หากพนักงานถามเรื่องอื่นที่ไม่เกี่ยวกับการลางาน ให้ตอบว่า "ขออภัยครับ ระบบนี้รองรับเฉพาะการสอบถามเรื่องสิทธิ์การลาเท่านั้นครับ"

[รูปแบบการตอบ (Output Format)]
ให้ตอบกลับตามโครงสร้างนี้เสมอ:
- ทักทายอย่างสุภาพ
- สรุปสิทธิ์ที่เหลืออยู่ (ถ้ามีการถาม)
- ตอบคำถามหลัก หรือ แจ้งผลการพิจารณาเบื้องต้น (ใช้ Bullet points เพื่อให้อ่านง่าย)
- แนะนำขั้นตอนต่อไป (Call to Action) เช่น "กรุณากดส่งคำขอในระบบ HRIS เพื่อให้หัวหน้าอนุมัติครับ"
```

## 2. Logic (Prompt Guard / Pseudo-code)
การพึ่งพา AI เพียงอย่างเดียวมีความเสี่ยง (Hallucination) เราจึงต้องใช้สถาปัตยกรรม Hybrid (Rule-based + AI) โดยนำ Code Logic มาครอบการทำงานของ AI ไว้ (Input/Output Guard) เพื่อดักจับความผิดปกติก่อนส่งให้พนักงาน
```JabaScript

function processLeaveRequest(userMessage, userLeaveBalance, daysNotice) {
    
    // ตรวจสอบกฎบริษัทด้วย Hard-code ก่อนส่งให้ AI เพื่อป้องกัน AI อนุมัติผิดพลาด
    if (daysNotice < 3) {
        return "ระบบไม่สามารถดำเนินการได้ เนื่องจากการลาพักร้อนต้องแจ้งล่วงหน้าอย่างน้อย 3 วันตามกฎบริษัทครับ";
    }

    if (userLeaveBalance <= 0 || userLeaveBalance > 15) {
        return "ระบบไม่สามารถดำเนินการได้ เนื่องจากโควตาวันลาของคุณไม่เพียงพอ หรือสิทธิ์สะสมเกิน 15 วันครับ";
    }

    // เรียกใช้ AI เพื่อสร้างคำตอบที่เป็นธรรมชาติ
    const prompt = buildPromptWithContext(userMessage, userLeaveBalance);
    const aiResponse = callLLM(prompt);

    // Output Guard
    if (aiResponse.includes("อนุมัติ") && !isAuthorizedManager) {
        console.error("AI Policy Violation Detected");
        return "คำขอของคุณถูกบันทึกแล้ว กรุณารอหัวหน้างานอนุมัติในระบบครับ";
    }

    return aiResponse;
}
```
